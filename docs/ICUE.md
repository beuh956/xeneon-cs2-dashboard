# Configuration iCUE XENEON EDGE

Utiliser le widget **iFrame**, pas le widget URL Web.

Code à coller :

```html
<iframe src="http://localhost:31982/" width="100%" height="100%" frameborder="0"></iframe>
```

Si localhost ne fonctionne pas, utiliser l'adresse réseau affichée par le serveur :

```html
<iframe src="http://192.168.1.xxx:31982/" width="100%" height="100%" frameborder="0"></iframe>
```
